## 1. 디스크 사용량 모니터링

### df (Disk Free) - 전체 현황
```bash
# 사람이 읽기 쉬운 형식
df -h

# 특정 마운트 포인트
df -h /home

# inode 사용량 (파일 개수 한계)
df -i

# JSON 형식 (파싱용)
df -h --output=source,size,used,avail,pcent,target
```

### du (Disk Usage) - 상세 분석
```bash
# 현재 디렉토리 요약
du -sh *

# 상위 10개 대용량 디렉토리
du -h /home --max-depth=1 | sort -hr | head -n 10

# 100MB 이상 파일 찾기
find / -size +100M -type f 2>/dev/null

# 특정 사용자 사용량
du -sh /home/ai
```

### ncdu - 인터랙티브 분석
```bash
# 설치
sudo apt install ncdu

# 스캔 (병렬 처리)
ncdu -t8 /home

# 화살표로 탐색, d로 삭제
```

## 2. inotify - 파일 변경 실시간 감지

### 개념
```
inotify = inode notify
- Linux 커널 2.6.13+ 내장
- 폴링 없이 즉시 감지
- CPU 효율적
```

### inotify-tools 설치
```bash
sudo apt install inotify-tools
```

### inotifywait - 변경 감지
```bash
# 단일 이벤트 감지 후 종료
inotifywait /home/ai

# 지속 모니터링 (-m)
inotifywait -m /home/ai

# 재귀적 모니터링 (-r)
inotifywait -m -r /home/ai

# 특정 이벤트만 (-e)
inotifywait -m -e create,delete,modify /home/ai

# 출력 형식 지정
inotifywait -m -r --format '%T %w%f %e' --timefmt '%Y-%m-%d %H:%M:%S' /home/ai
```

### 주요 이벤트
```
access    - 파일 읽음
modify    - 파일 수정됨
create    - 파일 생성됨
delete    - 파일 삭제됨
move      - 파일 이동됨
open      - 파일 열림
close     - 파일 닫힘
attrib    - 속성 변경 (권한 등)
```

### 스크립트 예시
```bash
#!/bin/bash
# 파일 변경 로그 수집

inotifywait -m -r -e create,delete,modify \
    --format '%T %w%f %e' \
    --timefmt '%Y-%m-%d %H:%M:%S' \
    /home/ai 2>/dev/null | while read line; do
    echo "$line" >> /var/log/file-changes.log
done
```

## 3. find - 시간 기반 파일 탐색

### 최근 수정 파일
```bash
# 최근 10분 내 수정된 파일
find /home -mmin -10

# 최근 1시간 내
find /home -mmin -60

# 최근 1일 내
find /home -mtime -1

# 최근 7일 내
find /home -mtime -7
```

### 대용량 파일
```bash
# 100MB 이상
find / -size +100M -type f 2>/dev/null

# 1GB 이상
find / -size +1G -type f 2>/dev/null

# 크기순 정렬
find / -size +100M -type f -exec ls -lh {} \; 2>/dev/null | sort -k5 -hr
```

## 5. 제한사항

### inotify 한계
```
- max_user_watches: 기본 8192개
- NFS/원격 파일시스템 미지원
- /proc, /sys 미지원
- 재부팅 시 재설정 필요
```

### 해결책
```bash
# watch 한계 증가
echo 524288 | sudo tee /proc/sys/fs/inotify/max_user_watches

# 영구 적용
echo 'fs.inotify.max_user_watches=524288' >> /etc/sysctl.conf
```
